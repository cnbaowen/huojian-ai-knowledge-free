# 变更记录

## 0.3.0-rc.3 - 2026-07-18

- 完善公开仓库首页，集中说明产品能力、适用场景、部署方式和使用边界。
- 优化贡献、安全和社区协作说明，使公开页面更便于使用者阅读和参与。
- 增加包含功能简介、部署步骤、系统截图、已知限制和双平台下载入口的版本说明。
- 调整自动发布流程，发布带版本说明、源码压缩包和 SHA256 校验文件的 GitHub 预发布版。

## 0.3.0-rc.2 - 2026-07-18

- 完善 GitHub 仓库首页、演示脚本和社区协作模板。
- 增加使用问题 Issue Form、Dependabot 配置、自动 Release 工作流和跨平台发布包脚本。
- 增加源码 ZIP 及 SHA256 自动生成流程。
- 修正安全漏洞 Issue 入口配置，避免使用无效的相对链接。

## 0.3.0-rc.1 - 2026-07-18

- 完善两级模块导航、顶部工具栏、页签系统、Logo 和全局视觉样式。
- 免费版导航改为严格白名单，只显示工作台、完整知识治理、企业微信机器人和必要系统设置。
- 登录页采用双栏产品展示结构，并使用独立免费版访问令牌。
- 使用真实浏览器验收知识库、内部问答、引用来源和企业微信机器人页面，控制台无错误和警告。

## 0.2.0-rc.1 - 2026-07-18

- 从轻量 RAG Demo 完善为具备完整产品界面的知识管理免费版。
- 增加工作台、独立知识分类、知识质检、RAG 评测和知识切片预览。
- 增加 DOCX、XLSX 和可复制文字 PDF 解析。
- 增加企业微信群机器人 Webhook 加密配置、地址校验和主动测试。
- 继续物理排除 AI客服、CRM、内容生产、情报中心、智能体和 OpenClaw。

## 0.1.0-rc.1 - 2026-07-18

- 建立独立免费版源码和全新 Git 历史。
- 打通文档上传、解析切片、本地向量检索、引用回答和反馈闭环。
- 增加 Apache-2.0 发布候选、Linux CI、安装文档、演示数据和安全模板。
- 增加 SQLite 本地验收与独立容器持久化验收。
