# 社区行为准则

## 我们的承诺

我们希望每个人都能在专业、友善和相互尊重的环境中参与项目，不因经验、身份、背景或观点不同而受到骚扰或歧视。

## 适用范围

本准则适用于仓库的 Issue、Pull Request、代码评审、Discussion、提交记录及其他与本项目有关的公开交流。

## 鼓励的行为

- 尊重不同经验、背景与观点。
- 就事论事，以事实和实际情况为依据，不攻击个人。
- 友善提出和接受建设性意见。
- 在发现错误后及时更正，帮助讨论回到问题本身。

## 不接受的行为

- 骚扰、歧视、威胁、侮辱、人身攻击或持续扰乱讨论。
- 未经允许公开他人的个人信息或私密沟通内容。
- 发布侵犯他人隐私、版权或其他合法权益的内容。
- 利用项目社区发布垃圾信息、恶意链接或与项目无关的广告。

## 执行方式

维护者可以隐藏或删除违反准则的内容，锁定或关闭相关讨论，并根据情况临时或永久限制参与。处理过程中会尽量保护相关人员的隐私。

涉及漏洞、凭据泄露或数据安全的问题，请按照 [安全政策](SECURITY.md) 私密报告，不要在公开 Issue 中披露可利用细节。
