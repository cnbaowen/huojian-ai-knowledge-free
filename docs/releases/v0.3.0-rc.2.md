# 火建AI知识管理免费版 v0.3.0-rc.2

这是面向 GitHub 首次公开发布准备的候选版本。它提供一套可独立部署、可真实运行的企业知识库闭环，而不是单页 RAG Demo。

## 主要能力

- 企业资料分类、上传、解析、切片、索引、下载和预览
- TXT、Markdown、LOG、CSV、JSON、DOCX、XLSX 和可复制文字 PDF
- 内部知识问答、知识不足拒答、引用来源和用户反馈
- 知识质检、质量评分和 RAG 问题评测
- 企业微信群机器人 Webhook 加密配置与受控测试
- 默认本地引用式回答，可选 OpenAI 兼容模型
- 商业版同款两级导航、顶部页签和产品视觉语言

## 独立与安全边界

- 独立 Git 历史、Docker Compose 项目、网络、数据库卷和文档卷
- 不包含 AI 客服、CRM、内容生产、情报中心、智能体、OpenClaw、商业许可证或系统升级
- 不包含客户数据、密钥、日志、备份或商业版运行配置

## 快速体验

```bash
cp .env.example .env
docker compose up -d --build
docker compose exec backend php artisan migrate --force
```

打开 `http://localhost:18080`，上传 `demo-data/company-travel-policy.md`，然后询问“差旅报销最晚什么时候提交？”。

## 发布状态

这是 Pre-release。公开部署前请更换全部默认密码和访问令牌，并将服务放置于内网、VPN 或受控反向代理之后。
