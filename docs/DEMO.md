# 三分钟演示脚本

本演示只使用仓库内虚构资料，不需要模型 API Key，也不会调用外部模型服务。

## 准备

按 [安装说明](INSTALL.md) 启动系统，打开 `http://localhost:18080`，输入 `.env` 中的 `FREE_API_TOKEN`。

## 演示步骤

1. 进入“知识治理 → 企业知识库”。
2. 点击“上传文档”，选择 `demo-data/company-travel-policy.md`。
3. 上传完成后确认状态为“已索引”，知识切片数大于 0。
4. 点击“查看”，确认可以预览切片内容和原始来源。
5. 进入“内部知识问答”。
6. 输入：`差旅报销最晚什么时候提交？`
7. 确认答案包含“每月 25 日前”，右侧至少出现一个引用来源。
8. 再输入仓库资料中不存在的问题，确认系统不会编造确定答案。
9. 进入“知识质检”和“RAG 评测”，展示知识治理与可重复验证能力。
10. 进入“企业微信 → 群机器人配置”，说明 Webhook 加密保存；不要在公开演示中填写真实 Webhook。

## 讲解重点

- 这不是一个单页聊天框，而是从资料管理到质量评测的完整知识库闭环。
- 回答必须绑定引用来源，方便员工核验。
- 默认本地模式无需外部模型密钥，适合离线体验。
- 免费版的代码、数据库和容器独立运行，不依赖外部业务系统。

## 演示安全

- 只上传虚构或已获授权公开的资料。
- 截图前检查页面中没有 Token、模型密钥、Webhook 和个人信息。
- 不在公网直接暴露后端 `18000` 端口。
